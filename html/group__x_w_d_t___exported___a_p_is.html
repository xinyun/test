Configurate The WatchDog Timer(WDT)'s Timer Interval. This function is to configure The WatchDog Timer(WDT)'s Timer Interval. There are three factors to determine the Timer Interval, they are:

  • clock source
  • Prescaler divider
  • reload value
Parameters
[in]ulBaseis the base address of the WatchDog Timer(WDT) module. This value must be xWDT_BASE.
[in]ulConfigis the WDT clock source Selection.
  • xWDT_S_INTSL
  • xWDT_S_EXTSL
[in]ulReloadis the WDT's reload value. 0xFF <= ulReload <= 0xFFFFFFFF
Returns
None.
Note
xWDTInit(ulBase, ulConfig, ulReload) must be called before function:
  • xWDTEnable and - xWDTFunctionEnable.
