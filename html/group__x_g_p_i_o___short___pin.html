Detailed Description

1. Where to use this group

The following values define the short pin argument(dShortPin) to several of the XPinTypexxx APIs and all the API which have a eShortPin argument. Such as xGPIOSPinRead() xGPIOSPinWrite()

2.CoX Port Details

* +-------------------------+----------------+------------------------+
* | xGPIO Short Pin ID | CoX | LPC17xx |
* |-------------------------|----------------|------------------------|
* | PXn | Mandatory | PA0 PA1 ... PA31 |
* | (X = A/B/...) | |------------------------|
* | (n = 0/1/...) | | PB0 PB1 ... PB31 |
* | | |------------------------|
* | | | ... |
* | | |------------------------|
* | | | PE0 PE1 ... PE31 |
* +-------------------------+----------------+------------------------+
