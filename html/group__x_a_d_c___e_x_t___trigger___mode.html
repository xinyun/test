Detailed Description

Can be passed to the ADCConfigure() as the ulTrigger parameter, when the trigger source set as xADC_TRIGGER_EXT_$ShortPin$. The ulTrigger parameter is an OR vaule with the xADC Trigger Source. such as xADC_TRIGGER_EXT_PD2 | xADC_TRIGGER_EXT_RISING_EDGE.
