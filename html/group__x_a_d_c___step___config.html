Detailed Description

ADC step config values, such as single-end / difference, channel source. They are can be passed to the xADCStepConfigure() as the ulConfig parameter.

xADC Step Config Divide

The Step Config value is an logic OR type of the following config:

  • xADC_CTL_CH0
  • xADC_CTL_CH1
  • xADC_CTL_CH2
  • xADC_CTL_CH3
  • xADC_CTL_CH4
  • xADC_CTL_CH5
  • xADC_CTL_CH6
  • xADC_CTL_CH7
