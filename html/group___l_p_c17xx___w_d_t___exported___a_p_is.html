void WDTCfg ( unsigned long  ulCfg,
unsigned long  ulValue 
)

Configurate The WatchDog Timer Module. This function is to configure The WatchDog Timer(WDT) clock source and mode.

Parameters
[in]ulConfigis the WDT clock source Selection and Mode. This value can be OR of two type of value:
  • clock source
    • WDT_CFG_CLKSRC_IRC
    • WDT_CFG_CLKSRC_APB
    • WDT_CFG_CLKSRC_RTC
  • clock source
    • WDT_CFG_INT_MODE
    • WDT_CFG_RESET_MODE
[in]ulValueis the Timer's reload value. 0xFF <= ulReload <= 0xFFFFFFFF
Returns
None.
Note
xWDTInit(ulBase, ulConfig, ulReload) must be called before function: xWDTEnable and xWDTFunctionEnable.

Parameters
[in]ulFlagsis WDT status flag. This parameter must be WDT_FLAG_TIMEOUT.
Returns
None.
Note
