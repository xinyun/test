#define xUARTCharGet (   ulBase)    UARTByteRead(ulBase)

Waits for a character from the specified port. Gets a character from the receive FIFO for the specified port. If there are no characters available, this function waits until a character is received before returning.

Parameters
[in]ulBaseis the base address of the UART port. This value can be one of the following value:
  • xUART0_BASE
  • xUART1_BASE
  • xUART2_BASE
  • xUART3_BASE
Returns
Returns the character read from the specified port, cast as a long.

#define xUARTCharGetNonBlocking (   ulBase)    UARTByteReadNoBlocking(ulBase)

Receives a character from the specified port. Gets a character from the receive FIFO for the specified port.

This function replaces the original UARTCharNonBlockingGet() API and performs the same actions. A macro is provided in uart.h to map the original API to this API.

Parameters
[in]ulBaseis the base address of the UART port. This value can be one of the following value:
  • xUART0_BASE
  • xUART1_BASE
  • xUART2_BASE
  • xUART3_BASE
Returns
Returns the character read from the specified port, cast as a long. A -1 is returned if there are no characters present in the receive FIFO. The UARTCharsAvail() function should be called before attempting to call this function.

#define xUARTCharPutNonBlocking (   ulBase,
  ucData 
)    UARTByteWriteNoBlocking(ulBase, ucData)

Sends a character to the specified port. Writes the character ucData to the transmit FIFO for the specified port. This function does not block, so if there is no space available, then a false is returned, and the application must retry the function later.

This function replaces the original UARTCharNonBlockingPut() API and performs the same actions. A macro is provided in uart.h to map the original API to this API.

Parameters
[in]ulBaseis the base address of the UART port. This value can be one of the following value:
  • xUART0_BASE
  • xUART1_BASE
  • xUART2_BASE
  • xUART3_BASE
[in]ucDatais the character to be transmitted.
Returns
Returns true if the character was successfully placed in the transmit FIFO or false if there was no space available in the transmit FIFO.

Sets the configuration of a UART. This function configures the UART for operation in the specified data format. The baud rate is provided in the ulBaud parameter and the data format in the ulConfig parameter.

Parameters
[in]ulBaseis the base address of the UART port. This value can be one of the following value:

  • xUART1_BASE
  • xUART2_BASE
  • xUART3_BASE
[in]ulBaudis the desired baud rate.
[in]ulConfigis the data format for the port (number of data bits, number of stop bits, and parity). The ulConfig parameter is the logical OR of three values: the number of data bits, the number of stop bits, and the parity.
  • xUART_CONFIG_WLEN_5
  • xUART_CONFIG_WLEN_6
  • xUART_CONFIG_WLEN_7
  • xUART_CONFIG_WLEN_8
  • xUART_CONFIG_STOP_1
  • xUART_CONFIG_STOP_2
  • xUART_CONFIG_PAR_NONE
  • xUART_CONFIG_PAR_EVEN
  • xUART_CONFIG_PAR_ODD
  • xUART_CONFIG_PAR_ONE
  • xUART_CONFIG_PAR_ZERO
Returns

#define xUARTIntDisable (   ulBase,
  ulIntFlags 
)    UARTIntDisable(ulBase, ulIntFlags)

Disables individual UART interrupt sources. Disables the indicated UART interrupt sources. Only the sources that are enabled can be reflected to the processor interrupt; disabled sources have no effect on the processor.

Parameters
[in]ulBaseis the base address of the UART port. This value can be one of the following value:
  • xUART0_BASE
  • xUART1_BASE
  • xUART2_BASE
  • xUART3_BASE
[in]ulIntFlagsis the bit mask of the interrupt sources to be disabled. this parameter is the logical OR of any of the following:
  • xUART_INT_ERROR All Error interrupt
  • xUART_INT_RT Receive Timeout interrupt
  • xUART_INT_TX Transmit interrupt
  • xUART_INT_RX Receive interrupt
Returns
None.

#define xUARTRxErrorGet (   ulBase)    UARTStatGet(ulBase)

Gets current receiver errors. This function returns the current state of each of the 4 receiver error sources. The returned errors are equivalent to the four error bits returned via the previous call to UARTCharGet() or UARTCharGetNonBlocking() with the exception that the overrun error is set immediately the overrun occurs rather than when a character is next read.

Parameters
[in]ulBaseis the base address of the UART port. This value can be one of the following value:
  • xUART0_BASE
  • xUART1_BASE
  • xUART2_BASE
  • xUART3_BASE
Returns
Returns a logical OR combination of the receiver error flags,
  • xUART_RXERROR_FRAMING
  • xUART_RXERROR_PARITY
  • xUART_RXERROR_BREAK
  • xUART_RXERROR_OVERRUN
