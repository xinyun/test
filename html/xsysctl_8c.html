void SysCtlPeripheralDisable2 ( unsigned long  ulPeripheralBase)

Disables a peripheral by base address.

Peripherals are Disabled with this function. At power-up, all peripherals are disabled; they must be enabled in order to operate or respond to register reads/writes.

Parameters
[in]ulPeripheralBasea Peripheral base indicate which peripheral to be enabled. it must be one of the following value:
  • ETH_BASE
  • DMA0_BASE
  • USB_BASE
  • GPIO_PORTA_BASE ... More information, please refer to LPC17xx LowLayer Peripheral Memmap.
Returns
None.
