void TimerCaptureCfg ( unsigned long  ulBase,
unsigned long  ulChs,
unsigned long  ulCfgs 
)

Configure timer capture function. This function is used to configure timer capture function.

Parameters
[in]ulBaseis the base address of the Timer port. This value can be one of the following value:
  • TIMER0_BASE, - TIMER1_BASE,
  • TIMER2_BASE, - TIMER3_BASE.
[in]ulChsis the timer channel. This value can be one of the following value:
  • TIMER_CAP_CH_0
  • TIMER_CAP_CH_1
[in]ulCfgsis the parameters for timer match mode. This value can be one of the following value:
  • TIMER_CFG_CAP_RISING Capture rising edge signal.
  • TIMER_CFG_CAP_FALLING Capture falling edge signal.
  • TIMER_CFG_CAP_INT Triggle interrupt when capture event occurs.
Returns
None.
Note
- TIMER_CFG_CAP_INT can be used with - TIMER_CFG_CAP_RISING and - TIMER_CFG_CAP_FALLING, i.e: TimerCaptureCfg(TIMER0_BASE, TIMER_CAP_CH_0, TIMER_CFG_CAP_RISING | TIMER_CFG_CAP_INT);

Configurate The Timer's mode and tick frequency. This function is to configurate The Timer's mode and tick frequency.

Parameters
[in]ulBaseis the base address of the Timer port. Can be one of the following value:
  • xTIMER0_BASE
  • xTIMER1_BASE
  • xTIMER2_BASE
  • xTIMER3_BASE
[in]ulChannelis the channel of the Timer port. Can be one of the following value:
  • xTIMER_CHANNEL0 is timer channel.
  • xTIMER_CHANNEL1 is capture channel.
[in]ulConfigis the mode Configuratation of the Timer port. Can be one of the following value:
  • Timer mode
    • xTIMER_MODE_ONESHOT
    • xTIMER_MODE_PERIODIC
    • xTIMER_MODE_CONTINUOUS
    • xTIMER_MODE_CAPTURE
  • Timer counter direction
    • xTIMER_COUNT_UP
[in]ulTickFreqis the tick frequency of the Timer port.
Returns
None.
Note
For LPC17xx, Only xTIMER_CHANNEL1 can be configure into Capture mode.
