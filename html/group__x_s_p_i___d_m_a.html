Detailed Description

Values that show xSPI DMA.

1. Where to use this group

Values that can be passed to xSPIDMAEnable and xSPIDMADisable as the ulDmaMode parameter.

2. CoX Port Details

xSPI DMA CoX LPC17xx
xSPI_DMA_TX Mandatory N
xSPI_DMA_RX Mandatory N
xSPI_DMA_BOTH Non-Mandatory N
Note
For LPC17xx, There is only one SPI module and not support DMA operation.
