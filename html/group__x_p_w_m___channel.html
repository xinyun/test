Detailed Description

Values that show xPWM Channel.

1. Where to use this group

Values that can be passed to all the function in xpwm.c as the ulChannel parameter.

PWM Event Channel Flag, Used by IntHandle's Event Callback Function as ulMsgParam parmeter. User Callback function can use this to detect what Channel event happened.

2.CoX Mandatory and CoX Non-mandatory
