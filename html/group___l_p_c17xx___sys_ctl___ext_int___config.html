Detailed Description

LPC17xx SysCtl External Interrupt Configuration.

Configure. The following are values that can be passed to the following function:

  • SysCtlExtIntCfg
  • SysCtlExtIntFlagGet
  • SysCtlExtIntFlagCheck
  • SysCtlExtIntFlagClear As the input/output parameter.
