Detailed Description

Values that show xPWM Event Type
.

1. Where to use this group

PWM Event/Error Flag, Used by IntHandle's Event Callback Function as ulEvent parameter. User Callback function can use this to detect what event happened.

2. CoX Mandatory and CoX Non-mandatory

xPWM Event Source CoX LPC17xx
xPWM_EVENT_PWM Mandatory Y
xPWM_EVENT_CAP non-Mandatory Y
