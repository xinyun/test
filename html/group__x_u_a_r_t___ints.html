Detailed Description

Values that show xUART Interrupt.

1. Where to use this group

Values that can be passed to xUARTIntEnable, xUARTIntDisable, and xUARTIntClear as the ulIntFlags parameter, and returned from UARTIntStatus.

2. CoX Port Details

xUART Interrupts CoX LPC17xx
xUART_INT_ERROR Mandatory Y
xUART_INT_RT Mandatory N
xUART_INT_TX Mandatory Y
xUART_INT_RX Mandatory Y
