Detailed Description

1. Where to use this group

Values that can be passed to xGPIODirModeSet as the ulPinIO parameter, and returned from xGPIODirModeGet.

2.CoX Port Details

This is the GPIO mode in CoX. It show as same Macro name in all series of CoX implementation.

xGPIO Dir Mode CoX LPC17xx
xGPIO_DIR_MODE_IN Mandatory Y
xGPIO_DIR_MODE_OUT Mandatory Y
xGPIO_DIR_MODE_HW Non-Mandatory N
xGPIO_DIR_MODE_QB Non-Mandatory N
xGPIO_DIR_MODE_OD Non-Mandatory Y

More please refer to xGPIO Dir Mode in MD file.
