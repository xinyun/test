Detailed Description

Values that show xUART Event Flags.

1. Where to use this group

Uart Event/Error Flag, Used by IntHandle's Event Callback Function as ulMsgParam parmeter. User Callback function can user this to detect what event happened.

2. CoX Port Details

xUART Events CoX LPC17xx
xUART_EVENT_TX Mandatory Y
xUART_EVENT_RX Mandatory Y
xUART_EVENT_OE Mandatory Y
xUART_EVENT_FE Mandatory Y
