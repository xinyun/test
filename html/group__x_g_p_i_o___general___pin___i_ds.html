Detailed Description

1. Where to use this group

The following values define the bit field for the ulPins argument to several of the APIs. So all the API which have a ulPins argument must use this group.
xGPIODirModeSet
xGPIODirModeGet
xGPIOPinIntCallbackInit
xGPIOPinIntEnable
xGPIOPinIntDisable
xGPIOPinIntClear
xGPIOPinRead
xGPIOPinWrite
xGPIOPadConfigSet
xGPIOPadConfigSet

2.CoX Port Details

This is a mandatory argument in cox,it show as xGPIO_PIN_n, n is general be 7\15 or 31. For LPC17xx n is 31.

xGPIO General Pin ID CoX LPC17xx
xGPIO_PIN_n Mandatory xGPIO_PIN_0
xGPIO_PIN_1
...
xGPIO_PIN_30
xGPIO_PIN_31

More please refer to xGPIO General Pin ID in MD file.
